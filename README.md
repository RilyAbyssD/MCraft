# Minecraft専用コミュニティー 「MCraft」

# 概要
Minecraftの専用コミュニティーであり、Minecraftに特化したサービスです。
サーバーのスタッフを募集することが可能だったり 自分の作品を共有 宣伝する事が出来ます。
